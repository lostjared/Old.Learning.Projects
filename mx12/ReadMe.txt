How to play
------

Simply use the arrow keys to guide yourself through the menus.
Using the return key will let you select an item.
When playing as the Hero, you can move him left,
right, with the arrow keys. Fire with S and jump with A.

Left Key = Move Left
Right Key = Move Right
A = Jump
S = Fire
