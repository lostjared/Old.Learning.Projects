Your first MasC++ program:

 #insert "console.msrc";

master MyProg {

uses {console;}

var { var$ str = "Hello "; }

begin {
print << master.str << " World\n";
}

end {
print << "Goodbye World\n";
call console.pause;
}
}

Will produce the following output:



To run the script execute:

masc++ myprog.msrc
